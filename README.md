# 🍽️ Restaurant Table Booking System

A web-based Restaurant Table Booking System developed using PHP and MySQL that allows customers to reserve tables online and helps restaurant administrators efficiently manage reservations.

## 📌 Project Overview

The Restaurant Table Booking System is designed to simplify restaurant reservation management. Customers can book tables online, check booking status, and view reservation details, while administrators can manage bookings through a dedicated dashboard.

---

## ✨ Features

### 👨‍💼 Admin Features
- Secure Admin Login
- Dashboard Overview
- Manage Table Reservations
- Approve/Reject Bookings
- View Booking History
- Manage Customers
- Generate Reports
- Change Password

### 👨‍👩‍👧 Customer Features
- Online Table Reservation
- Check Reservation Status

## ⚙️ Installation Guide

### Prerequisites
- PHP 7.4 or above
- MySQL
- XAMPP/WAMP/LAMP
- Web Browser

### Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/Restaurant-Table-Booking-System.git

2. Move the project folder to:
xampp/htdocs/

3. Start Apache and MySQL.

4.Create a database:
CREATE DATABASE rtbsdb;

5. Import the SQL file into the database.

6. Configure database settings in:
   includes/config.php

7.Open browser and run:
http://localhost/Restaurant-Table-Booking-System
